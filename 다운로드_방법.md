# 📥 GitHub에서 생성된 이미지 다운로드 방법

## 🎯 다운로드 위치 2가지

GitHub Actions로 실행하면 **2곳**에서 이미지를 다운로드할 수 있습니다!

---

## 방법 1: Artifacts (추천! ⭐)

### 1단계: Actions 탭으로 이동
1. GitHub 레포지토리 메인 페이지
2. 상단 메뉴에서 **Actions** 탭 클릭

### 2단계: 완료된 워크플로우 선택
- 워크플로우 목록에서 **초록색 체크마크** ✅ 있는 항목 클릭
- 또는 **노란색 원** 🟡이면 실행 중 (기다려야 함)
- **빨간색 X** ❌면 실패 (로그 확인 필요)

### 3단계: Artifacts 다운로드
- 페이지 하단으로 스크롤
- **Artifacts** 섹션 찾기
- 두 가지 중 하나 다운로드:
  - `급등이슈_이미지` (ZIP 파일)
  - `등락률상위_이미지` (ZIP 파일)

### 4단계: ZIP 압축 해제
- 다운로드한 ZIP 파일 압축 풀기
- PNG 이미지 파일 확인

**보관 기간:** 30일 (이후 자동 삭제)

---

## 방법 2: 레포지토리에 자동 커밋

### 설정되어 있는 기능:
생성된 이미지가 자동으로 `output` 폴더에 커밋됩니다!

### 다운로드 방법:
1. GitHub 레포지토리 메인 페이지
2. **output** 폴더 클릭
3. 최신 이미지 파일 클릭
   - `급등이슈_20251204_1.png`
   - `등락률상위_20251204.png`
4. **Download** 버튼 클릭 또는 **Raw** 버튼 → 우클릭 → 다른 이름으로 저장

**장점:** 
- ✅ 영구 보관 (삭제 안 됨)
- ✅ 이전 날짜 이미지도 모두 확인 가능
- ✅ Git 히스토리로 변경 내역 추적

**단점:**
- ❌ 레포지토리 용량 증가
- ❌ 많은 이미지가 쌓이면 정리 필요

## 🖼️ 실제 다운로드 화면 예시

### Actions 탭 화면
```
Actions
├── 급등이슈 카드 자동 생성
│   └── ✅ Update: 2025.12.04  (클릭!)
│       └── Summary
│           └── Artifacts (하단)
│               └── 📦 급등이슈_이미지 (Download)
│
└── 등락률 상위 자동 생성
    └── ✅ Update: 2025.12.04  (클릭!)
        └── Summary
            └── Artifacts (하단)
                └── 📦 등락률상위_이미지 (Download)
```

### 레포지토리 output 폴더
```
output/
├── 급등이슈_20251201_1.png
├── 급등이슈_20251201_2.png
├── 급등이슈_20251204_1.png   ← 최신!
├── 급등이슈_20251204_2.png   ← 최신!
├── 등락률상위_20251201.png
└── 등락률상위_20251204.png   ← 최신!
```

## 💡 어느 방법을 사용할까?

### Artifacts 추천 (방법 1) - 이런 경우
- ✅ 오늘 생성한 이미지만 필요
- ✅ 레포지토리 용량 절약
- ✅ 자동으로 30일 후 삭제되어도 OK

### 레포지토리 커밋 추천 (방법 2) - 이런 경우
- ✅ 이미지를 영구 보관하고 싶음
- ✅ 이전 날짜 이미지도 확인하고 싶음
- ✅ Git 히스토리로 관리하고 싶음

## 🔧 자동 커밋 기능 끄려면?

워크플로우 파일에서 마지막 단계 삭제:

```yaml
# 이 부분을 삭제하거나 주석 처리

#   if: success()
#   run: |
#     git config --local user.email "action@github.com"
#     ...
```

## 📱 모바일에서 다운로드

**GitHub 앱 사용 시:**
1. Actions 탭 → 완료된 워크플로우
2. Artifacts는 **모바일 앱에서 다운 불가** 😢
3. **대신 output 폴더 이용** → 파일 클릭 → 저장

**해결방법:**
- PC에서 다운로드하거나
- output 폴더 이미지를 모바일에서 다운

## ❓ FAQ

### Q: Artifacts가 안 보여요
A: 워크플로우가 **성공(✅)**했는지 확인. 실패하면 Artifacts 생성 안 됨.

### Q: 30일 지나면 어떻게 되나요?
A: Artifacts는 자동 삭제. output 폴더 이미지는 영구 보관.

### Q: 다운로드 횟수 제한 있나요?
A: 없습니다! 무제한 다운로드 가능.

### Q: ZIP 말고 PNG만 바로 다운받을 수 없나요?
A: Artifacts는 항상 ZIP. PNG만 받으려면 output 폴더 이용.
